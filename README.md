# Regulated Power Supply Design

This project involves the design of a regulated DC power supply using rectification, filtering, and voltage regulation techniques.

## Overview

The system converts AC input to a stable DC output and analyzes performance under varying load conditions.

## Key Features

- AC to DC conversion
- Full-wave rectifier design
- Filtering to reduce ripple
- Voltage regulation using Zener diode

## Repository Structure

- `docs/` – design report and analysis  
- `images/` – circuit diagrams and measurements  

## Skills Demonstrated

- Power electronics fundamentals
- Circuit design and analysis
- Load regulation and ripple analysis
- Debugging and iteration
